{{< /blocks/cover >}} {{% blocks/lead color="primary" %}} Git Profile Manager (GPM) is a simple command-line tool to make switching projects or identities with Git easier. GPM lets you quickly apply any number of Git configuration commands repeatably, perfect for things like changing emails, signing keys or command behaviour when changing between projects. {{% /blocks/lead %}} {{< blocks/section color="dark" >}} {{% blocks/feature icon="fa-tools" title="Simple and Reliable" url="/docs/getting-started/profiles/" %}} Profiles are stored in easy-to-edit text files, and applied using Git itself. {{% /blocks/feature %}} {{% blocks/feature icon="fas fa-cloud-download-alt" title="Easy to Install" url="/docs/getting-started/installation/" %}} Available as native packages for common platforms as well as single binaries so you can install it anywhere. {{% /blocks/feature %}} {{% blocks/feature icon="fab fa-github" title="Contributions welcome!" url="https://github.com/agc93/git-profile-manager" %}} GPM is fully open-source and open to community contributions and fixes! {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section >}}
